ReadWriteLock

java.util.concurrent.locks
Interface ReadWriteLock

所有已知实现类：
ReentrantReadWriteLock

public interface ReadWriteLock

A ReadWriteLock维护一对关联的locks ，一个用于只读操作，一个用于写入。 read lock可以由多个阅读器线程同时进行，只要没有作者。 write lock是独家的

为什么要使用:

希望更加细腻度的控制锁,读和写锁分离
