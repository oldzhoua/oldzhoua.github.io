最后发现,在多线程操作的时候,报了ConcurrentModificationException异常 ( 翻译过来就是 并发修改异常 )

解决办法:

三种:

第一种: 采用集合工具类提供的安全方法Collections.synchronizedList()

第二种是使用List 的子类 Vector 是支持同步的

第三种是使用CopyOnWriteArrayList类

结果就是 :不报异常
